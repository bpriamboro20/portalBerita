Perkenalan diri saat interview


Selamat siang bu, terima kasih banyak telah memberikan saya kesempatan interview pada hari ini.
Saya Bayu Aji Priamboro lulusan baru dari universitas indraprasta PGRI Jakarta jurusan teknik informatika.

Saat ini, saya memfokuskan diri saya untuk menjadi seorang front-end web developer.

Saya memiliki pengalaman bekerja sebagai pramuniaga di spbu pertamina dan sebagai officeboy  di PT.Violad Creative Studio.

Saya juga pernah memiliki pengalaman magang di PMB Bidan Lena sebagai programmer dalam pembuatan aplikasi dekstop sebagai penunjang operasional.

Kegiatan yang saya lakukan saat ini selain aktif mencari pekerjaan saya juga seorang freelancer, saya adalah seorang driver ojek online (Grab) dan Event organizer crew di duta venue. Selain itu saya juga aktif mengeksplore diri saya di dunia perkodingan ini dengan mengikuti kursus maupun belajar otodidak baik dari youtube ataupun membaca dokumentasi.

Alasan saya melamar posisi sebagai administrator database di perusahaan ini adalah karena saya tertarik dengan reputasi perusahaan dalam mengembangkan produk berkualitas tinggi.

Saya percaya saya dapat menyumbang kreatifitas di dalam project yang menarik di perusahaan ini. Serta dapat belajar dari tim yang berpengalaman. 

Saya mampu berkolaborasi di dalam tim. Saya senang bekerja sama dengan orang lain dan mendengarkan ide2 dari mereka. 

Kemampuan komunikasi saya juga cukup kuat, saya mampu menyampaikan ide saya dengan jelas dan memastikan bahwa semua orang memahaminya. 

Selain itu saya juga pandai dalam pemecahan masalah. Saya bisa menghadapi tantangan dengan tenang, menganalisis situasi dan menemukan solusi yang efektif.

Saya yakin dengan kombinasu dari 3 kelebihan saya ini memungkinkan saya untuk memberikan kontribusi yang maksimal di dalam tim.

Kemampuan-kemampuan yang telah saya sebutkan sebelumnya, telah saya aplikasikan saat saya melakukan magang di PMB bidan lena


Saya sadar kelemahan saya adalah seorang pelupa. Saya kadang lupa pada hal-hal tertentu. Namun saya sedang berusahan mengatasi dengan menggunakan reminder atau todolist untuk mencatat hal-hal penting.
